Types of attachments

TypeEnglish (EN)العربية (AR)Best for
Deep link A URL pointing to a specific product page, checkout, or sign-up form. When users click, they land directly on that page. عنوان URL يشير إلى صفحة منتج محددة أو سجل الدفع أو نموذج التسجيل. عندما ينقر المستخدمون، يصلون مباشرة إلى تلك الصفحة. Single product, specific offer
Product set A collection of 1–30 products shown as a carousel or grid. Users swipe to explore before clicking to buy. مجموعة من 1-30 منتج معروضة كحامل أو شبكة. يمرر المستخدمون للاستكشاف قبل النقر للشراء. Multiple products, catalog showcase
Web view A custom landing page that opens inside Snapchat without leaving the app. Faster, lower friction, higher CTR. "صفحة هبوط مخصصة تُفتح داخل سناب شات بدون مغادرة التطبيق. أسرع وأقل احتكاكاً ومعدل نقر مرتفع أعلى." High-converting offers, mobile-optimized

How to set up

StepEnglish (EN)العربية (AR)
For deep link In Ads Manager > Create Ad > Attachment > Select "Deep Link". Paste your product URL. Set up tracking parameters (UTM) if needed. Save. في Ads Manager > إنشاء إعلان > Attachment > اختر "Deep Link". الصق عنوان URL للمنتج. قم بإعداد معاملات التتبع (UTM) إذا لزم الأمر. احفظ.
For product set Go to Catalog Manager > Create Product Set > Select products > Publish. Then in Ad Manager > Attachment > Select Product Set. The carousel preview shows in your ad. انتقل إلى Catalog Manager > Create Product Set > اختر المنتجات > Publish. ثم في Ad Manager > Attachment > اختر Product Set. يظهر معاينة الحامل في إعلانك.
For web view Build a mobile-optimized landing page (forms, checkout, etc). In Ad Manager > Attachment > Web View > Paste URL. Test on mobile. بناء صفحة هبوط محسّنة للأجهزة المحمولة (نماذج وسجل دفع وما إلى ذلك). في Ad Manager > Attachment > Web View > الصق عنوان URL. اختبر على الهاتف المحمول.

Best practices

PracticeEnglish (EN)العربية (AR)
Match ad to attachment If your ad says "Shop now", the attachment should show the product grid, not your homepage. Consistency = higher conversion. إذا كان إعلانك يقول "تسوق الآن"، يجب أن يعرض الملحق شبكة المنتجات وليس صفحتك الرئيسية. التسق = تحويل أعلى.
Mobile-optimize everything All attachments load on mobile. Test your deep link and web view on actual phones – slow load = abandoned clicks. جميع المرفقات تُحمّل على الهاتف المحمول. اختبر الرابط العميق وعرض الويب على الهواتف الفعلية - التحميل البطيء = نقرات مهجورة.
Use UTM parameters Add ?utm_campaign=snapchat&utm_source=deep_link to your URLs so you can track which attachments drive conversions in Google Analytics. أضف ?utm_campaign=snapchat&utm_source=deep_link إلى عناوين URL الخاصة بك بحيث يمكنك تتبع المرفقات التي تدفع التحويلات في Google Analytics."
A/B test attachments Run one ad set with deep link vs. another with product set. Measure CTR, CPA, and conversion rate. Scale the winner. شغّل مجموعة إعلانات واحدة بـ deep link مقابل أخرى مع مجموعة المنتج. قس CTR وتكلفة الاكتساب ومعدل التحويل. وسّع الفائز.
Attachments are one of the highest-leverage optimizations. Type alone (deep link vs. product set) can swing conversion rates 20–30%.
